# Database Modeling

Use Wordpress built-in database

```sql
CREATE TABLE Assets (
	ASSET_ID INTEGER NOT NULL PRIMARY KEY AUTOINCREMENT,
	ASSET_NAME TEXT NOT NULL,
	HOLDING_AMOUNT NUMERIC NOT NULL,
	CREATED_DATE TEXT NOT NULL,
	DELETED_DATE INTEGER,
	CATEGORY TEXT,
	CONSTRAINT Assets_PK PRIMARY KEY (ASSET_ID)
);
```
