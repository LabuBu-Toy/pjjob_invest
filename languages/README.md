# Language files directory

This directory contains translation files for the PJJob Invest plugin.

To add translations:

1. Create a `.po` (Portable Object) file for your language
2. Use the text domain `pjjob-invest`
3. Place the file in this directory

File naming convention:
- `pjjob-invest-{language_code}.po` (editable)
- `pjjob-invest-{language_code}.mo` (compiled)

Example: `pjjob-invest-es_ES.po` for Spanish

## Supported Language Codes

- `en_US` - English (United States)
- `es_ES` - Spanish (Spain)
- `fr_FR` - French (France)
- `de_DE` - German (Germany)
- And any other language supported by WordPress

For more information, visit https://developer.wordpress.org/plugins/internationalization/
